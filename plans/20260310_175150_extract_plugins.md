## Goal
Extract claude, codex, copilot plugins into external repos installable via `bab add`.
Keep opencode as the only bundled plugin (ships with bab).

## Bundled vs installed plugin rules
- Plugin IDs are globally unique across bundled + installed
- `bab add` fails if manifest ID matches a bundled plugin
- Bundled plugins are listable but not removable via `bab remove`
- Loader discovers bundled plugins first, then installed plugins
- No override behavior
