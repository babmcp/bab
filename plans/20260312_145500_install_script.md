# Plan: Bash install script

## Goal

One-liner install: `curl -fsSL https://raw.githubusercontent.com/babmcp/bab/main/install.sh | bash`

With args: `curl -fsSL ... | bash -s -- --prefix /usr/local/bin`

## Strategy: binary-first, brew fallback

1. Download pre-built binary from GitHub releases (fast, no deps)
2. If download fails and `brew` is available → `brew install babmcp/tap/bab`
3. If neither works → error with manual instructions

## Depends on

- Release workflow (`plans/20260312_143000_selfupdate_command.md`) — needs published binaries + `checksums.sha256`
- Homebrew tap (`plans/20260312_144500_homebrew_tap.md`) — for the fallback path

## Script flow

```
install.sh [--prefix DIR]
```

1. **Detect platform**: `uname -s` → darwin/linux; `uname -m` → arm64/x86_64 (map x86_64 → x64, aarch64 → arm64)
   - Unsupported platform (Windows/MINGW/CYGWIN) → error "Unsupported platform. See https://github.com/babmcp/bab for manual install."
2. **Resolve latest version**: `curl` GitHub API `/repos/babmcp/bab/releases/latest` → extract tag
3. **Download binary**: `curl -fSL` the matching asset `bab-{platform}-{arch}`
4. **Download checksums**: fetch `checksums.sha256` from same release
5. **Verify checksum**: platform-aware:
   - macOS: `shasum -a 256 -c`
   - Linux: `sha256sum -c`
   - Mismatch → abort with error
6. **Strip macOS quarantine**: on darwin, `xattr -d com.apple.quarantine <binary>` (ignore errors)
7. **Detect existing install**: if target path exists, check if brew-managed (`readlink` → contains `Cellar`). If so, warn "Existing Homebrew install detected. Run `brew upgrade bab` instead or use --force to overwrite."
8. **Install**: `chmod +x` → move to `~/.local/bin/bab` (or `--prefix` if given)
   - Create install dir if it doesn't exist
9. **PATH check**: if install dir not in `$PATH`, print shell profile instructions
10. **Verify**: run `bab --version`
11. **Print update instructions**: "To update later, run: `bab selfupdate`"

### Brew fallback (on binary download failure)

- Check `command -v brew`
- If available: `brew install babmcp/tap/bab`
- Print clearly: "Installed via Homebrew. To update, run: `brew upgrade bab`"
- If brew not available: print error with manual install options (GitHub release URL, brew tap instructions)

## Install location

- Default: `~/.local/bin/bab` (no sudo needed)
- Override: `--prefix /usr/local/bin` (may need sudo)
